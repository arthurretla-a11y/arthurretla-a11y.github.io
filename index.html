📨 Configuration EmailJS

Comment ça marche ?
EmailJS permet d'envoyer les messages du formulaire directement dans ta boîte mail, gratuitement, sans serveur.

Étapes :
1. Crée un compte sur emailjs.com
2. Ajoute un service email (Gmail, Outlook…) → copie le Service ID
3. Crée un template avec les variables {{from_name}}, {{from_email}}, {{message}}, {{to_email}} → copie le Template ID
4. Dans Account → API Keys → copie ta Public Key
5. Colle les 3 valeurs ci-dessous et sauvegarde ✓

Les messages du formulaire de contact seront affichés dans le dashboard Messages.
